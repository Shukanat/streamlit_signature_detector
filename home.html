Cette application a été dévéloppée pour tester la détéction de signatures sur les différents documents. Elle est écrite entièrement en Python à l'aide de Streamlit. Pour la détection de signature, l'application utilise l'algorithme Yolo v4 pré-entraîné sur le jeu de données publique Tobacco-800.

Auteurs: Vitali Shchutski & Cédric Soares
